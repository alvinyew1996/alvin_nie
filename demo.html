我们之间的故事

——献给那个曾经让我笑、让我成长的人

🎭 项目特色

📖 3D破旧日记本

使用Three.js创建的逼真3D书本模型，从侧面移动到正面的开场动画

🎬 现实翻页效果

模拟真实书本的翻页动画，配合粒子特效和尘粒飘散

🎵 背景音乐系统

宫崎骏、杨丞琳、陈华、张碧晨的精选音乐，完美配合每个章节

✨ 特殊视觉效果

星空、粒子、彩虹、泡泡、流星等浪漫特效

📱 响应式设计

完美适配各种设备屏幕，移动端友好

🎥 视频播放支持

集成视频播放和音频控制，支持静音和音量调节

📚 章节预览

封面

3D书本从侧面移动到正面，星空背景和月亮幸运草装饰，回访用户特殊消息

第一章：初遇

四张照片记录初次相遇，翻页动画效果，杨丞琳《雨爱》背景音乐

第二章：相处的日子

24个页面展示日常生活，多种照片布局（拼贴、网格、单张），特殊标注和特效

第三章：旅行

奏折式横向滚动效果，19个页面记录旅行回忆，视频和照片混合展示

第四章：那些笑容

特殊照片特效展示，重要视频播放，彩虹泡泡浪漫特效

第五章：我想对你说

语音消息播放，礼物照片拼贴展示，张碧晨《骗》背景音乐

第六章：结尾

书本合上动画，星空特效，温馨寄语文字动画

📋 使用说明

  1. 安装依赖：运行 npm install 安装项目依赖
  2. 添加媒体文件：将照片、视频、音频文件放入对应文件夹
  3. 启动开发服务器：运行 npm run dev 启动项目
  4. 自定义内容：修改 chapter-generator.js 中的数据结构
  5. 构建生产版本：运行 npm run build 构建项目
